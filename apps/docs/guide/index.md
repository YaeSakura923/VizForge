# Getting Started

## Prerequisites

- Node.js >= 18
- Python >= 3.10 (for backend)
- Docker + docker-compose (for deployment)

## Installation

```bash
git clone https://github.com/YaeSakura923/vtk-visualization-pipeline.git
cd vtk-visualization-pipeline
npm install
```

## Development

```bash
npm run dev
```

Opens the Studio app at `http://localhost:5173`.

## Build

```bash
npm run build
```

Output is in `apps/studio/dist/`.
